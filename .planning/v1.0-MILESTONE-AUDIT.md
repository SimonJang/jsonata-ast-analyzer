### Integration Finding: ADV-02 Composed Edge Case (Non-Critical)

**Severity:** Non-critical (tech debt)
**Location:** `src/walker.ts` `walkPath()` variable-resolution branch

When `walkPath` resolves a variable step and returns early via the variable-resolution path, `walkFilterStages` is never called. If the `VariableNode` carries a `.predicate` with an unresolvable `$variable`, the ADV-02 `[*]` emission silently does not fire.

| Case | Behavior |
|------|----------|
| `orders[$field].price` (direct NameNode) | ADV-02 fires correctly → `orders[*]` emitted |
| `($data := orders; $data[$field].price)` (resolved VariableNode) | ADV-02 does NOT fire → `orders[*]` NOT emitted |

No test covers the broken case. All 101 tests pass.
